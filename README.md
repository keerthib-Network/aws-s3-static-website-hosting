# AWS S3 Static Website Hosting

## Project Overview

This project demonstrates how to host a static website using Amazon S3. The website is publicly accessible through the S3 Static Website Endpoint.

## AWS Services Used

- Amazon S3
- Static Website Hosting
- Bucket Policy

## Features

- Created an Amazon S3 Bucket
- Uploaded HTML files
- Enabled Static Website Hosting
- Configured Bucket Policy for public access
- Hosted a static website

## Project Architecture

Internet
↓
S3 Website Endpoint
↓
Amazon S3 Bucket
↓
index.html

## Technologies

- AWS S3
- HTML

## Outcome

Successfully deployed a static website using Amazon S3 and made it publicly accessible through the S3 website endpoint.
